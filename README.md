# book-store

> A Vue.js project
This application build up using Vuejs + Nodejs + Mongodb.

#Build Setup

Clone this Repository.

#Install dependencies
npm install

#Serve with hot reload at localhost:8888
npm run dev

#For server at localhost:2000
cd api/

npm install

nodejs server.js
